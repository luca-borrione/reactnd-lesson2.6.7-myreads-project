(private, inner) search(query, resolveopt, rejectopt) → {Array.<TBook>}

Promise executor that performs a search to find books matching the query string typed by the user. If no books are found, or if the search completes with an error, then an empty array is returned so that the list under the searchbar is cleared. The promise is rejected if an exception is triggered while performng the search.

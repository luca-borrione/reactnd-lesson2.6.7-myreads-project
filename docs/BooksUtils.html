BooksUtils

BooksUtils

Provides a set of utility functions that helps dealing with the literal book objects handled in the BooksAPI in a consistent way across the app.

Methods

(static) getShelfTitle(shelf) → {string}

Returns the title of a shelf, lowercasing the noise words and capitalising the non noise words, based on the shelf id as it appears in the book literal objects fetched using the BooksAPI

Example
// returns: Want to Read
const title = getShelfTitle('wantToRead');
Parameters:
Name Type Description
shelf string

The shelf unique name
Throws:

Throws an exception if an unexpected shelf id has been passed

Type
Error
Returns:

The shelf title with the expected text cases

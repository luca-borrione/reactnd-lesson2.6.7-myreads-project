ArrayUtils

Provides a set of array utility functions.

Methods

(inner) diff(…arrays) → {Array}

Retrieves the symmetric difference between arrays.

Example
const a = ['a', 'd', 'e'];
const b = ['a', 'b', 'c', 'd'];
diff(a, b); // (3) ["e", "b", "c"]
Parameters:
Name Type Attributes Description
arrays Array <repeatable>

the arrays to compare
Returns:
  • the symmetric difference between the given arrays
Type
Array

(inner) diffByKey(key, …arrays) → {Array}

Retrieves the symmetric difference between arrays of objects by comparing a key in common between the objects.

Example
const a = [{k:1}, {k:2}, {k:3}];
const b = [{k:1}, {k:4}, {k:5}, {k:6}];
const c = [{k:3}, {k:5}, {k:7}];
diffByKey('k', a, b, c); // (4) [{k:2}, {k:4}, {k:6}, {k:7}]
Parameters:
Name Type Attributes Description
key string

the key in common between the objects to use in the comparison
arrays Array <repeatable>

the arrays to compare
Returns:
  • the symmetric difference between the given arrays
Type
Array

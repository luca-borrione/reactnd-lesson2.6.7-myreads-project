App

App

When first loaded it automatically fetches all the books in the shelves and it stores them internally as a state.
This becomes the source of truth across the different sections of the app.

(private) fetchAllBooks() → {Promise}

Asynchronously retrieves the list of all the books currently in a shelf from remote, then it stores them internally in the state as booksInShelves collection.

getBookShelf(bookID) → {TShelfKey}

Retrieves the current shelf associated with a book, based on a given book id. If the book is not present in the booksInShelves collection the shelf will be NONE.

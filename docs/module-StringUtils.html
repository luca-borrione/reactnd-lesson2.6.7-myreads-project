StringUtils

Provides a set of string utility functions.

Methods

(inner) makeTitle(str) → {string}

Uppercases the first character of every word of a given string, except if considered noise words, such as prepositions, which are kept lowercased.
All the other characers are lowercased by default.

It also makes sure to capitalise:

  • the first word of the string
  • the first word after a column even if they are considered noise.
Example
// returns: The Complete Guide: The New and the Old
const title = makeTitle('the complete guide: the new and the old');

(inner) ucfirst(str) → {string}

Uppercases the first character of the first word of a given string.
All the other characers are lowercased by default.

Example
